# Пр1

## Требования

* JDK 17 или выше

## Запуск проекта

### Linux / macOS

```bash
java -cp lib/aspectjrt.jar:out Main
```

### Windows (cmd / PowerShell)

```cmd
java -cp lib\aspectjrt.jar;out Main
```

Программа запустит консольное меню с возможностью добавления, удаления, сравнения элементов и логирования методов через AspectJ.
